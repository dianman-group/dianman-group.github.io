元器件如何选型

电路 电阻 电感 电容 磁珠
Publish Date:   2021-07-04

本博文指在探讨在硬件电路开发中，如何对电阻、电容、电感、磁珠的选型。该元器件是组成电路的基本单元。

电阻

导体对电流的阻碍作用就叫该导体的电阻。电阻（Resistor，通常用“R”表示）是一个物理量，在物理学中表示导体对电流阻碍作用的大小。导体的电阻越大，表示导体对电流的阻碍作用越大。不同的导体，电阻一般不同，电阻是导体本身的一种性质。导体的电阻通常用字母R表示，电阻的单位是欧姆，简称欧，符号为Ω。

如果说芯片是电路的骨架，那么电阻就是在芯片之间起连接作用的关节。电阻的阻值、放置等，对设计的成功起着重要的作用。在电路设计时，对于电阻有四个关注点：

  1. 电阻的阻值
  2. 电阻的尺寸
  3. 电阻的额定功率；
  4. 电阻的精度

电阻使用时需要注意的：

  1. 在功耗高的路径上，如果串联了电阻，选型时需要考虑电阻的额定功率。
  2. 当电阻用于设定器件的工作参数时，应选择高精度电阻。
  3. 巧妙使用0R电阻。

电容

电容（Capacitance）亦称作“电容量”，是指在给定电位差下自由电荷的储藏量，记为C，国际单位是法拉（F）。一般来说，电荷在电场中会受力而移动，当导体之间有了介质，则阻碍了电荷移动而使得电荷累积在导体上，造成电荷的累积储存，储存的电荷量则称为电容。

电容在电路的贡献非常大，即便是在一块普通的板子上，电容也是随处可见。因此电容在电路设计中是最为常见的器件。但同时电容也是最容易被忽视的器件，在设计中，设计者们往往知道需要放置电容，但是对电容的选型却是随心所欲。

电容的作用：

通交隔直、旁路、耦合、滤波、补偿、充放电、储能

在高速电路中电容器不再是纯粹的电容，而是带有电阻、电感等成分的小电路。其中ESL（等效串联电感）由电容器件的引脚电感和电容器两极间等效电感串联而成。其中ESR（等效串联电阻）是电容器的引脚电阻和电容器两极间等效电阻构成。主要取决于电容工作温度、工作频率以及电容本身的导线电阻等。 Rleak（Leakage Curremt）则取决于电容器本身特有的泄露特性.

ESR在电路中的作用：

  1. 对于高速电路中，往往希望电容的阻抗越小越好；一方面，对于高频信号的交流耦合，电容串联在高频信号上，目的是隔断高频信号收发两端的直流分量，同事又希望高频信号的衰减越小越好，如果电容的ESR较大，对于交流耦合的高频分量，相当于在信号中串联了一个不小的电阻，将产生一定的信号衰减。另一方面并联在电源和地之间的滤波电容，其作用是将噪声等干扰信号提供一个极低的阻抗回路，ESR较大的电容显然不发满足这个要求。在高速电路中即使ESR很小的电容仍然无法满足低阻抗的要求，往往需要并联多个电容，最大程度降低ESD。

  2. 根据电容损耗角正切值的定义，较大的ESR会产生较大的损耗功率P，如果大到一定程度，且该类型电容较多时，就不能不考虑电容上的损耗。

  3. 不过也有例外，有些LDO就需要ESR大一点的电容，其原理是在LDO电源的负载电流发生瞬间变化时，利用ESR能立即产生电压波动，从而引起LDO电源反馈电路的动作，以便LDO电源针对负载变化做出快速的调整。

陶瓷电容：

高速电路设计中，常用到的陶瓷电容种类有NPO、X7R、X5R和Y5V。他们的区别主要在于填充的介质不同，从而引起温度、电压稳定性的不同。
NPO是温度补偿型陶瓷电容，是电容量最稳定的一种陶瓷电容。工作温度范围是-55°C~+125°C；
X7R表示工作温度范围在-55°C~+125°C，温度稳定性为±15%的电容。
X5R表示工作温度范围在-55°C~+85°C，温度稳定性为±15%的电容。
Y5V表示工作温度范围在-30°C~+85°C，温度稳定性为+22%~-82%的电容。

注意：在高频电路中，推荐使用NPO，X7R，X5R，尽量不要使用Y5V。

钽电容：

钽电容使用金属钽作为介质，具有温度特性好、ESL值小、高频滤波效果好、体积小、节省PCB面积、容值较大等特点。因此钽电容一般被应用在需要大容量电容滤波的场合，如为CPU滤波等高耗能器件。
钽电容的缺点是耐电压和耐电流能力较弱。一般钽电容的工作电压相对额定电压要降额50%以上。遇到以下三种情况需要降额70%以上使用：

  1. 负载呈现较强的感性；
  2. 串联电阻较小
  3. 瞬变电流比较大
    钽电容主要滤除低频段噪声，陶瓷电容滤除高频段噪声。从成本上来看钽电容的价格正比于容值和额定电压的乘积，在使用大容值的钽电容时，还需要增加电压降额的比例，这就必然造成成本的增加。设计中往往使用多个小的钽电容并联以实现大容值钽电容相同的容值。这样做，一方面有利于设计的可靠性，一方面也有利于成本的降低。在涉及热插拔的电源滤波电路上，尽量避免使用钽电容。

铝电解电容：

铝电解电容使用电解液作为介质，外壳的铝制圆筒作为负极，内部插入一块金属板作为正极。铝电容容量大、耐压高，但温度稳定性差、精度差、高频滤波性能差，仅适用于低频滤波。

钽电容不适用应用于较大瞬变电流的场合，而在这种场合下，就需要用到铝电解电容。铝电解电容在使用时需要降额至少20%使用。

铝电容一般都是插件式或者是SMD贴片式，因此ESL和ESR比较大，同时由于采用液体作为介质，在极高温和极低温环境下，性能也极其不稳定。

铝电解电容的ESR值比常温下的值可能大好几倍，滤波性能大大降低。并且随着产品的老化铝电解电容内部的电解液降逐渐干涸，容量逐渐减小，ESR逐渐增大，滤波效果减弱。所以在高速电路中应尽量避免使用铝电解电容。

OSCON电容：

OSCON电容并不是一种电容类型，而是SANYO公司的一种性能比较好的电解电容品牌。

OSCON电容的外形酷似铝电解电容，其优点在于，OSCON电容ESR较小，温度稳定性相对电解电容要好，价格相对钽电容较低，缺点是绝大多数OSCON电容的体积比较大。在DCDC电路中往往在输入和输出端需要放置容量大的钽电容，成本较高，这种情况下可以使用OSCON电容进行替换。

去耦电容：

作用是为保证器件稳定工作而为器件电源提供的本地“小池塘”。另一个作用是为高速运行器件产生的高频噪声提供一个就近流入地面的路径，避免影响其他负载。

旁路电容：

作用是为前级提供一条流入地平面的低阻抗路径，避免这些干扰影响正在高速工作的器件。

电感

电感是闭合回路的一种属性，是一个物理量。当电流通过线圈后，在线圈中形成磁场感应，感应磁场又会产生感应电流来抵制通过线圈中的电流。这种电流与线圈名。它是描述由于线圈电流变化，在本线圈中或在另一线圈中引起感应电动势效应的电路参数。电感是自感和互感的总称。提供电感的器件称为电感器。

电感的作用：

1、 通交流，隔直流
根据Z = jwL;频率越高，电感阻抗越大。
2、 阻碍电流的变化，保持器件工作电流的稳定
当电流变化时，线圈感应出电动势，以产生与电流方向相反的感应电流，阻碍这种电流的变化。
3、 滤波
低通滤波器往往基于电感和电容构建。

电感选型需要注意：

  1. 电感值
  2. 直流电阻（DCR）
  3. 自谐振频率（Q最大时的频率）
  4. 额定电流

高频电感：

  1. 电感值范围在：0.6~390nH
  2. 直流电阻：一般而言吗，电感值越大，其对应的直流电阻也越大。
  3. 自谐振频率：可以达到12GHZ。电感值越大其对应的直流电阻也越大。
  4. 额定电流：几十毫安~几百毫安，电感值越大往往对应的额定电流越小。
    在应用中应选择谐振频率点高于工作频率的电感
一般信号使用电感：
  1. 电感值：0.01~1000uH
  2. 直流电阻：一般信号使用的电感直流电感会比高频信号使用的电感大一些。最小的直流电阻一般为100毫欧左右，大的可达到几欧姆。
  3. 自谐振频率：几十兆赫兹到几百兆赫兹，电感值越大，其对应的自谐振频率越小。
  4. 额定电流：几毫安到几十毫安，电感值越大，其对应的额定电流越小

注意： 电感当工作在低于谐振频率时，电感值基本保持不变，但工作频率超过谐振频率时，电感值会先增大，达到一定频率后将迅速减小

电感使用时需要注意：

  1. 当工作频率高于谐振频率时，电感器件表现出电容性，阻抗随着频率的升高而减小。
  2. 在应用中应使用谐振频率点高于工作频率的电感
  3. 高速电路设计中板间互连信号，纹波比板内信号大，可以使用一般信号用电感加以滤波。
  4. 高频电感和一般信号用电感额定电流都比较小，而直流电阻都比较大，不建议使用电源滤波
电源用电感：
  1. 电感值：1~470uH
  2. 直流电阻：电感值越大其对应的直流电阻越大。最小的一般为几毫欧，大的有几欧姆。
  3. 自谐振频率：几十兆赫兹到几百赫兹。电感值越大，对应的自谐振频率越小。

电源电路选择电感时需要注意：

  1. 电感工作频率高于谐振频率时，则电感值将会随着工作频率的升高而急剧减小，逐步呈现电容性。
  2. 工作电流要小于额定电流，当工作电流大于额定电流是未必会坏，但是电感值可能低于标称值。

总结：

  1. 高频电路中应该重点考虑其Q频率特性曲线，电源滤波使用电感考察其直流电阻、额定电流等参数。
  2. 电感和电容构成低通滤波器时，需要防止噪声频率点和谐振频率重合，以免共振。谐振频率F = 1/（2π√LC）
  3. 电感串联在电源电路中，需考虑电感器件的压降。

磁珠

磁珠专用于抑制信号线、电源线上的高频噪声和尖峰干扰，还具有吸收静电脉冲的能力。磁珠是用来吸收超高频信号，像一些RF电路，PLL，振荡电路，含超高频存储器电路（DDR SDRAM，RAMBUS等）都需要在电源输入部分加磁珠，而电感是一种蓄能元件，用在LC振荡电路，中低频的滤波电路等，其应用频率范围很少超过50MHZ。 磁珠有很高的电阻率和磁导率，等效于电阻和电感串联，但电阻值和电感值都随频率变化。

磁珠主要作用是吸收电源、信号上的噪声。相比于电感和电容，电容本身可以起到滤波的作用，电感和电容相配合也能起到滤波的作用，但这中滤波不能将噪声消除。电容的滤波是给信号提供一个低阻抗的回路，以便泄放到地平面，对于电感和电容构成的低通滤波器将低频信号泄放到地平面，对于高频信号相当于一个很大的电阻，将信号反射回去。

磁珠与电感和电容不同的是，低于转换点时反射噪声，高于转换点时吸收噪声。

以100R的为例，磁珠的阻抗Z由电阻成分R和电抗成分X共同决定。低频段X起主导作用，磁珠主要体现为电感性，功能是反射噪声；在高频段，R起到决定作用，磁珠主要体现为电阻性，功能是吸收噪声并将噪声转换成热。
转换点所在频率以下呈现电感性，转换点以上呈现电阻性，电感的作用就是反射噪声，电阻的作用就是噪声并转化为热能
转换点频率越高，磁珠体现电感性的频带越宽，对于低频的噪声吸收能力越弱，转换点所在频率越低，磁珠体现电阻的频带越宽，对低频噪声的吸收能力越强。

磁珠选型需要注意：

  1. 对于串了磁珠的线路，磁珠的转换点频率越低，线路震荡和波形失真就越小；反之则越大。
  2. 磁珠的工作电流不能超过额定电流，设计中需要考虑由于直流电阻的存在而造成的压降。
  3. 当工作频率高于谐振频率时，磁珠表现为电容性。
  4. 磁珠的转换点频率和谐振频率，在意义上有所不同
  5. 磁珠的选型需要注意额定电流、直流电阻、谐振频率

电感和磁珠的比较：

  • 不同点：
    1. 电感滤波是将电能转换成磁能：一种是转换回电能表现为噪声，另一种是向外辐射表现为EMI。磁珠是将信号转换为热能。
    2. 电感低频滤波比较好一般是50Mhz以内，磁珠其电阻能充分的吸收高频噪声，转换为热能并消除的目的
    3. 从EMC角度来看磁珠将高频噪声转换为热能，具有非常好的抗辐射能力。常应用于接口信号，单板高速信号滤波。
    4. 电感和电容都是储能，两者配合容易产生自激振荡。磁珠是耗能元件，与电容协同工作，不会产生自激。
    5. 电感一般用于大电流，磁珠用于小电流
    6. 磁珠和电感都有DCR；磁珠的DCR相对于同性能的电感更小一点，压降更小。
  • 相同点：
    • 额定电流：
      • 电感超过额定电流未必会损坏
      • 磁珠超过额定电流会对磁珠有损伤
    • 直流电阻：
      • 电感和磁珠都具有DCR选型时需要注意
    • 频率特性曲线
      • 电感和磁珠都会有这些参考曲线，以便选择合适的器件。
